MINI PROJECT


“REAL TIME FACE MASK DETECTION AND RECOGNITION USING OPEN CV “by using python

The prediction of Developed a real-time face mask detection and recognition 
system using OpenCV and Python programming language. The system utilized 
computer vision techniques to detect and recognize whether a person is wearing a 
face mask or not. Key responsibilities and achievements include: Gathered a 
diverse dataset of images, Preprocessed the collected images, Split the dataset into 
training and testing sets, and annotated the images to label mask-wearing status.
